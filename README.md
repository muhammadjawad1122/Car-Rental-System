# Car Rental System (Java)

Java-based Car Rental System developed as a semester OOP project.  
Manages cars, customers, and bookings with GUI and database support.

## Key Features
- Admin Login (authorized access only)
- Add/View Cars (model, brand, year, price)
- Add/View Customers (register/manage info)
- Book a Car (select car, customer, date, duration)
- View Bookings (all bookings in table format)
- Validation & Error Handling (prevent wrong data entry)

## Technologies
- Java | JavaFX (GUI) | MySQL | JDBC | Eclipse IDE

## How to Run
1. Open in Eclipse IDE  
2. Set up MySQL database and tables  
3. Update JDBC credentials  
4. Run main class
